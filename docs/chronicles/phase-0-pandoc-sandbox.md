# Phase 0: Pandoc Template Sandbox

## Entry 01: Project setup and planning (2026-03-06)

**What**: Established tantamount.rodeo as the sandbox for the `simple-markdown` pandoc template. Documented goals, structure, and mimeo integration requirements.

**Why**: Mimeo needs a zero-Node deployment option. This repo proves out pandoc + GitHub Actions as the build pipeline before integrating into mimeo proper.

**How**:

- Initial commit with placeholder index.html (default mimeo site)
- PLAN.md written with full template structure, build workflow, and mimeo integration spec
- CLAUDE.md added with project context and interoperability notes for Eleventy templates
- Project tracking initialized (docs/ structure)
