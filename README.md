# tantamount.rodeo

Sandbox for the `simple-markdown` template type - a pandoc-based single-page site with no Node.js or SSG toolchain.

## Build

```bash
pandoc index.md -o index.html --css style.css --standalone -f gfm
```

## Deploy

Push to `main` - GitHub Actions handles the rest (~20s total).
